Note

A 70 KB Windows note-taking app. No cloud, no account, no telemetry. One thing, done well.

Download Note View source on GitHub

Why Note exists

Existing note-taking apps have become heavy, talkative, intrusive. They want your email, your cloud, your usage statistics. They notify you, sync you, suggest features you never asked for.

Note takes the opposite path. A single window, anchored to the bottom right of your screen, always visible when you need it, minimizable when you don't. You write, the app saves, you close it. Tomorrow your notes are there, exactly where you left them.

What it does

  • 70 KB executable, single file
  • Always visible above other windows
  • Multiple independent sheets
  • Continuous automatic saving
  • Recycle bin with restore
  • Customizable theme, font size, keyboard shortcuts
  • Optional launch at Windows startup

What it doesn't do

  • No cloud sync
  • No account, no email, no signup
  • No telemetry, no analytics
  • No internet connection required, ever
  • No ads, no premium tier, no notifications
  • No automatic updates that change the UI every six months
